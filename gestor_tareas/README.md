 # Gestor de Tareas

Una aplicación de escritorio para gestionar tareas diarias, desarrollada con Python y Tkinter.

## Características
- Añadir nuevas tareas.
- Marcar tareas como completadas.
- Eliminar tareas.
- Interfaz gráfica intuitiva.

## Requisitos
- Python 3.6 o superior.
